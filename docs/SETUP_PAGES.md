# GitHub Pages setup

このPWAは静的ファイルだけで動くため、GitHub Pagesで公開できます。

## 設定手順

1. GitHubでこのリポジトリを開く。
2. Settings を開く。
3. Pages を開く。
4. Build and deployment の Source を Deploy from a branch にする。
5. Branch を main、folder を / root にする。
6. Save を押す。

数分後、Pages URLが表示されます。

## 注意

- `data/database.json` が正本データです。
- `src/styles.css` のA4横レイアウトは v5.9.6_LOCK として固定します。
- PDF化はブラウザの印刷から A4 横向きで出力します。
