# Event search policy

大会検索は、正式名称だけを要求しない。

## 方針

- 正式名称
- 略称
- 開催地
- 年
- 種目種別
- よく使う呼び方

を検索対象に含める。

例:

- 沼田
- 沼田町
- 沼田ベンチ
- 北海道ベンチ
- ベンチプレス
- 2026

## 安全方針

この検索は大会選択のユーザー体験を改善するためのものであり、選手履歴の確認済み昇格には使わない。

確認済み履歴は、従来通り公式結果PDF、HPA、JPA、IPF/Goodlift、APF/AAPF等の高信頼ソースに基づく。

A4印刷レイアウト v5.9.6_LOCK は変更しない。
