# Future event mode

今後の方向性メモです。

現在のPWAは、沼田町ベンチプレス大会のエントリー表を起点に、選手データを検索、保存、出力、コメント化しています。

将来的には、新しい大会情報とエントリー表が発表された段階で、その大会単位の解説資料を作れるアプリへ拡張します。

例:

- 2026-10-18 to 2026-10-19
- 旭川市
- 北海道秋パワーリフティング大会

## 将来必要になる機能

- 大会マスター
- エントリー表インポート
- 大会別Lot管理
- 大会別PDF出力
- 選手DBと大会エントリーDBの分離
- 大会ごとの読み上げコメント生成
