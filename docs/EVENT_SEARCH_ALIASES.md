# Event search aliases

大会検索では、正式名称以外の自然な呼び方も検索対象にする。

## 対象例

- 大会正式名称
- 略称
- 開催地名
- 年
- 種目名
- 主催・地域名
- ユーザーが普段使う呼び方

## 例: 沼田町ベンチプレス大会

検索で拾いたい語:

- 沼田
- 沼田町
- 沼田ベンチ
- 北海道ベンチ
- 北海道BP
- ベンチプレス
- BP
- 2026

## 注意

これは大会検索のUX改善であり、選手履歴の確認済み昇格ロジックではない。

確認済み履歴は公式結果PDF、HPA、JPA、IPF/Goodlift、APF/AAPFなどの高信頼ソースで確認する。
